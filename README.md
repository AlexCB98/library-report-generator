# Library Report Generator (Python File Handling)

A simple Python project that reads book titles from a text file, generates a library report, and creates individual files for each book.

This project was built using Python file handling, loops, string manipulation, and dynamic file generation.

---

## Features

* Read book titles from a text file
* Generate a library report automatically
* Count the total number of books
* Create individual files for each book
* Generate files dynamically using book names

---

## What I Practiced

* Reading files with `readlines()`
* Writing files with `write()`
* Working with directories and file paths
* Using loops and counters
* String manipulation with `strip()`
* Dynamic file creation using f-strings

---

```

main.py
```
